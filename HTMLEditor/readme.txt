【概要】
@HTMLEditorは、HTMLの基本的なタグ(<p>や<h1>など)を簡単に挿入できるシンプルなHTML作成支援ソフト(テキストエディタ)です。

【実行方法】
@HTMLEditor2.6.1.exe をダブルクリックして実行してください。インストールは不要です。

【必要な環境】
本ソフトウェアは .NET Framework 4.7.2 で開発されています。
そのため、実行には .NET Framework 4.7.2 以上がインストールされている必要があります。お持ちでない場合は、Microsoftの公式サイトからダウンロードしてご利用ください。

【動作チェック環境】
このソフトウェアは以下のOSで動作確認を行いました。
・Windows 7
・Windows 8/8.1
・Windows 10
・Windows 11
ARM CPUにおける動作は確認していないため、サポートや動作に関する保証はできません。

【アンインストール】
レジストリは使用していません。実行ファイル及び「readme.txt」「ヘルプガイド.html」「ショートカットキーマニュアル.html」「Scintilla.NET.dll」「runtimes」「backup」を削除するだけで問題ありません。
